ToString()

Returns the string representation of the current instance using default formatting.

public override readonly string ToString()

Returns

string

The string representation of the current instance.

Remarks

This method returns a string in which each element of the vector is formatted using the "G" (general) format string and the formatting conventions of the current thread culture. The "<" and ">" characters are used to begin and end the string, and the current culture's NumberGroupSeparator property followed by a space is used to separate each element.
