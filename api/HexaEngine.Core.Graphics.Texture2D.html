Reload()

Reloads the texture from the associated asset.

public void Reload()

Remarks

This method reloads the texture from the associated asset, disposing the previous texture and creating new resources according to the asset's path. The texture will be ready for use in rendering, shader resource, or unordered access. The TextureReloaded event is raised upon successful reloading.

Resize(IGraphicsDevice, Format, int, int, int, int, CpuAccessFlags, GpuAccessFlags, ResourceMiscFlag)

Resizes the texture with the specified parameters.

public void Resize(IGraphicsDevice device, Format format, int width, int height, int arraySize, int mipLevels, CpuAccessFlags cpuAccessFlags, GpuAccessFlags gpuAccessFlags = GpuAccessFlags.Read, ResourceMiscFlag miscFlag = ResourceMiscFlag.None)

Parameters

device IGraphicsDevice

The graphics device used to create the resized texture.

format Format

The format of the resized texture.

width int

The width of the resized texture.

height int

The height of the resized texture.

arraySize int

The array size of the resized texture.

mipLevels int

The number of mip levels for the resized texture.

cpuAccessFlags CpuAccessFlags

The CPU access flags for the resized texture.

gpuAccessFlags GpuAccessFlags

The GPU access flags for the resized texture (default is GpuAccessFlags.Read).

miscFlag ResourceMiscFlag

The miscellaneous flags for the resized texture (default is ResourceMiscFlag.None).

Remarks

This method resizes the texture to the specified dimensions with the provided access flags and format. If CPU and GPU access flags conflict or GPU access flags are used for reading, appropriate bindings and usage are adjusted. After resizing, the previous texture is disposed, and new resources are created accordingly. The texture will be ready to use for rendering, shader resource, or unordered access.
